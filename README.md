# Motorola-Sales-Dashboard
This project is an interactive Power BI dashboard developed to analyse Motorola mobile sales data across different cities, products, and time periods.    The aim of this project is to turn raw sales data into clear and meaningful insights that help understand business performance and customer behavior.
## Objective
The main objectives of this project are:
- To track overall sales performance
- To analyse sales by city, product, and brand
- To understand customer purchasing patterns
- To identify trends over time (daily and monthly)
- To evaluate customer satisfaction through ratings
## Tools & Technologies
- Power BI (Dashboard & Visualization)
- Power Query (Data Cleaning & Transformation)
- Excel / CSV (Data Source)
## Key Metrics
